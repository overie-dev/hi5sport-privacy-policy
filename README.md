_Last updated: May 4, 2026_

Hi5Sport (“we”, “our”, or “us”) respects your privacy and is committed to protecting your personal data.

This Privacy Policy explains how we collect, use, and protect your information when you use the Hi5Sport mobile application.

We may collect the following types of data:

a. Personal Information
Email address (via authentication)
User profile information (name, preferences, sports data)
b. Location Data
Approximate and/or precise location
Used to suggest nearby users, activities, and events
c. Health & Activity Data
Steps count
Distance traveled
Calories burned
Activity type (running, walking, cycling, etc.)
d. Device Information
Device type
OS version
App usage data

We use collected data to:

Provide and improve app features
Track and analyze sports activities
Enable social features (chat, groups, challenges, PickMeUp)
Suggest nearby users and activities
Send notifications (activities, messages, reminders)
Ensure app security and performance

3. Data Sharing

We do not sell your personal data.

We may share data with:

Google Firebase Services:
Authentication
Firestore (database)
Cloud Messaging (notifications)
Analytics

These services process data securely under Google’s privacy standards.

Data is stored securely via Firebase (Google Cloud)

Access to data is restricted and protected
